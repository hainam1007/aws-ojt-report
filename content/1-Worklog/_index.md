During the internship I built a foundation in AWS Cloud and gradually worked toward a capstone project:  
an **AWS Serverless AI Platform for Smart Accommodation Search**.

Typically, my weekly contents are organized as follows:

**Week 1:** [Getting familiar with AWS and core services](1.1-week1/)  
Introduction to the AWS console, IAM, regions and AZs, basic EC2 and S3 usage.

**Week 2:** [Networking and security fundamentals](1.2-week2/)  
VPC, subnets, security groups, NACLs, key security best practices.

**Week 3:** [Compute and storage deep dive](1.3-week3/)  
EC2 types, EBS, S3 advanced features (lifecycle, versioning), cost-awareness.

**Week 4:** [Relational databases on AWS](1.4-week4/)  
Amazon RDS basics, backup/restore, high availability options.

**Week 5:** [Data processing and analytics](1.5-week5/)  
Introduction to AWS Glue, Athena, and data lake concepts.

**Week 6:** [Visualization with Amazon QuickSight](1.6-week6/)  
Building dashboards and reports from curated datasets.

**Week 7:** [Serverless foundations](1.7-week7/)  
AWS Lambda, API Gateway, event‑driven patterns, basic monitoring with CloudWatch.

**Week 8:** [Serverless patterns for web backends](1.8-week8/)  
Designing simple APIs and integrating with databases in a serverless way.

**Week 9:** [Designing the AWS Serverless AI Platform](1.9-week9/)  
Defining requirements, architecture diagram, and data model for the capstone project.

**Week 10:** [Implementing backend and data layer](1.10-week10/)  
Building Lambda functions, APIs, and data storage for accommodation information.

**Week 11:** [Integrating AI recommendations](1.11-week11/)  
Connecting to an AI/ML model to rank and recommend accommodations based on user preferences.

**Week 12:** [Hardening, monitoring, and cost optimization](1.12-week12/)  
Improving security, adding logging/metrics, and reviewing cost.

**Week 13:** [Final review and documentation](1.13-week13/)  
Writing documentation, polishing the demo, and reflecting on lessons learned from the whole internship.
