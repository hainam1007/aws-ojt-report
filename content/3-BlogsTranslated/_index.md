This section lists and briefly introduces the AWS blogs you have summarized and translated.

### [Blog 1 – Build agentic systems with CrewAI and Amazon Bedrock](3.1-Blog1/)
This blog explains how to build multi‑agent (“agentic”) applications using the open‑source CrewAI framework together with Amazon Bedrock. It introduces key concepts such as agents, tools, tasks, and flows, shows how to orchestrate multiple specialized agents for complex workflows, and presents a reference solution for cloud‑security posture management on AWS using Bedrock models, Agents, and Knowledge Bases.

### [Blog 2 – Accelerate your AWS migration with AWS Training and Certification](3.2-Blog2/)
This blog focuses on how AWS Training and Certification supports each phase of a cloud‑migration journey: Assess, Mobilize, and Migrate & Modernize. It highlights common challenges (leadership alignment, migration strategy, lack of cloud skills) and maps them to specific learning resources and programs—such as AWS Technical Essentials, Migration Essentials, Knowledge Badges, and Skill Builder—to help organizations build the right capabilities and sustain long‑term success on AWS.

### [Blog 3 – Reserve your seat: Microsoft workloads on AWS sessions at re:Invent 2024](3.3-Blog3/)
This blog is a guide to re:Invent 2024 sessions dedicated to Microsoft workloads on AWS, including Windows Server, SQL Server, Active Directory, and .NET. It groups sessions by format (breakout sessions, builders’ sessions, chalk talks, code talks, and workshops), summarizes what attendees can learn—from cost optimization and licensing to modernization and generative‑AI use cases—and encourages readers to reserve seats early and visit the Microsoft workloads area in the expo.
